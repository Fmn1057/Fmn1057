## Proyectos destacados

### [Bencinas Chile · Proyecti-gasolinera](https://github.com/Fmn1057/Proyecti-gasolinera)

App web responsive para consultar precios de combustible en estaciones cercanas en Chile, consumiendo la API oficial de la **Comisión Nacional de Energía (CNE)**. Geolocalización, filtros por tipo de combustible, mapa con **Leaflet** + **OpenStreetMap** y renovación automática de token con refresh proactivo y reactivo.

`Node.js` · `Express` · `Leaflet` · `OpenStreetMap` · `API CNE`

### [Organizador de Escritorio](https://github.com/Fmn1057/Organizador-escritorio)

App de escritorio Windows con **casillas flotantes** que organizan los archivos del escritorio. Drag & drop, snap entre casillas, paleta de colores derivada del wallpaper y **comandos en lenguaje natural** vía Ollama local (`llama3.2`) con un esquema JSON estricto de acciones.

`Python` · `PyQt5` · `Ollama` · `Win32 API`

### [Sistema de Gestión de Alertas](https://github.com/Fmn1057/Resolucion-de-alertas)

Herramienta de escritorio que estandariza el flujo de tratamiento de alertas de monitoreo: pega texto bruto, devuelve **Short Description / Classification / Comment** listos para el ticket. Base de plantillas editable en Excel, lectura de correos Outlook `.msg`, reportes en PowerPoint y asistencia opcional con Ollama (con detección de *safety refusals* y reintento).

`Python` · `tkinter` · `pandas` · `openpyxl` · `python-pptx` · `Ollama`

### [South Park · Página web](https://github.com/Fmn1057/Southpark-pagina)

Sitio estático que consume mi fork de la **API-South-Park** a través de un túnel Cloudflare. Catálogo completo: personajes, episodios, especiales, música, juegos, familias, grupos, alter egos y Chinpokomon. Sin framework, un script por sección, configuración central en un solo archivo.

`HTML` · `CSS` · `JavaScript` · `Cloudflare Tunnel`
